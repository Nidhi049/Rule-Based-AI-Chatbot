#  Rule-Based AI Chatbot

A simple AI chatbot built with Python that responds to user inputs using if-else statement logic and pattern matching.


# About

This chatbot uses rule-based logic to understand and respond to user messages. It's a great beginner project to understand how chatbots work under the hood — without machine learning!


# Features

- Responds to common user inputs and questions
- Rule-based if-else decision making
- Simple and easy to understand code
- Runs directly in the terminal


# Tech Stack

- Language: Python 3
- Logic: If-else statements / pattern matching
- Interface: Command Line (CLI)


#How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Nidhi049/Rule-Based-AI-Chatbot.git
   cd Rule-Based-AI-Chatbot
   ```

1. Run the chatbot:
   bash
   python chatbot.py
   

1. Start chatting: Type your message and press Enter.


# Example Interaction

You: Hello
Bot: Hi there! How can I help you?

You: What is your name?
Bot: I am a Rule-Based AI Chatbot!

You: bye
Bot: Goodbye! Have a great day!
